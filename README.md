# 💼 My Portfolio

A personal portfolio website showcasing my projects, skills, and experience.  
Built with **HTML, CSS, and JavaScript**, and deployed using **GitHub Pages**.

## 🔗 Live Demo
👉 [View Portfolio](https://manishgupta5815.github.io/my-portfolio/)

## ✨ Features
- Responsive design for desktop and mobile
- Project showcase with links to GitHub repositories
- Contact section for quick reach
- Smooth animations and interactive UI

## 📂 Tech Stack
- HTML5  
- CSS3  
- JavaScript (ES6)  

## 🚀 Deployment
Hosted on **GitHub Pages** for free and easy access.
